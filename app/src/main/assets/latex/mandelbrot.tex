\text{The Mandelbrot set is the set of complex}\\
\text{numbers \(c\) for which the function}\\
\[
z_{n+1} = z_n^2 + c
\]\\
\text{does not diverge when iterated from \(z_0 = 0\).}\\
\text{Points that remain bounded are in the set,}\\
\text{while the colors represent how quickly}\\
\text{divergent points escape to infinity.}\\
\text{This boundary reveals an infinite, self-similar}\\
\text{complexity, a cornerstone of fractal geometry.}
