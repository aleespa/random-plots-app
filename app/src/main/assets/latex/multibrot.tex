\text{Multibrot sets are fractal sets that}\\
\text{generalize the Mandelbrot set by using}\\
\text{a power \(d\) other than 2 in the iterative}\\
\text{formula:}\\
\[ z_{n+1} = z_n^d + c \]\\
\text{The Mandelbrot set is the special case}\\
\text{where \(d = 2\). For larger integer values of}\\
\text{\(d\), the set exhibits \(d-1\)-fold rotational}\\
\text{symmetry. When \(d\) is not an integer, the}\\
\text{set becomes even more complex. In this app,}\\
\text{you can explore Multibrots with powers \(d\)}\\
\text{ranging from 2 to 10.}
