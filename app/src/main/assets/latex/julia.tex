\text{The Julia set for a complex number \(c\) is}\\
\text{the set of points \(z_0\) such that the sequence}\\
\[
z_{n+1} = z_n^2 + c
\]\\
\text{does not diverge to infinity. Unlike the}\\
\text{Mandelbrot set, where \(c\) varies, in a Julia}\\
\text{set, \(c\) is fixed and the initial value \(z_0\)}\\
\text{is varied across the complex plane.}\\
\text{Each value of \(c\) produces a unique and}\\
\text{intricate fractal shape, ranging from}\\
\text{connected "dust" to swirling "dragons".}
